# (사)한국중장년고용협회 (KAPAE 5070) 웹사이트 분석 결과

## 1. 개요 및 프로젝트 목적
본 프로젝트는 **(사)한국중장년고용협회(KAPAE 5070)**의 공식 웹사이트입니다. 중장년의 고용 시스템 구축, 일자리 창출, 그리고 AI 기술을 활용한 '워크그래피(Workgraphy)' 및 퍼플넷(Purplenet) 연동 서비스를 소개하고 제공하는 플랫폼 역할을 합니다.

- **도메인**: `kapae5070.org` (CNAME 설정됨)
- **주요 타겟**: 재취업, 전직, 창업 등을 준비하는 40대 이상 중장년층 및 중장년 고용을 원하는 기업.

## 2. 기술 스택 (Tech Stack)
프레임워크 없이 순수 웹 기술과 외부 BaaS(Backend as a Service)를 조합하여 가볍고 빠르게 동작하도록 설계되었습니다.

- **Frontend**: Vanilla HTML5, CSS3, ES6 JavaScript
  - 커스텀 CSS 변수(Design Tokens)를 활용한 체계적인 스타일링 (Navy, Gold, Purple 컬러 테마).
  - 프론트엔드 라우팅 없이 다중 HTML 파일(`index.html`, `notice.html`)로 구성.
  - 별도의 CSS/JS 파일로 모듈화 진행 (예: `app.js`, `journey_section_v2.js`).
- **Backend (Database)**: **Supabase**
  - REST API 형태로 직접 호출하여 공지사항/게시판 데이터 연동 및 문의사항 접수.
- **Mail Service**: **Resend** 연동
  - 문의 접수 폼 제출 시 Cloudflare Worker(`autumn-haze-7016.kimhyunjin0730.workers.dev`)를 경유하여 관리자에게 알림 메일 발송.

## 3. 핵심 아키텍처 및 주요 기능 분석

### 3.1. 메인 랜딩 페이지 (`index.html`)
사이트의 얼굴로 협회의 정체성 및 핵심 사업을 전달합니다.
- **섹션 구성**: 히어로(Hero), 협회 소개, 7대 핵심사업, 비전 및 목표, 퍼플넷(Purplenet) 제휴 랜딩, 연혁, 알림마당 요약표, 문의하기(Inquiry) 폼.
- **UI/UX 특징**: `IntersectionObserver`를 활용한 스크롤 리빌(Scroll Reveal) 애니메이션.
- **플로팅 챗봇**: 우측 하단에 고정된 챗봇이 하드코딩된 규칙 기반(Rule-based Knowledge Base)으로 협회/퍼플넷 관련 기본 상담 수행.

### 3.2. 워크그래피 AI 생성 마법사 (WGW - `app.js`)
가장 핵심적인 개인화 기능으로, 사용자의 고용 정보를 구조화된 데이터(Workgraphy)로 변환해 줍니다.
- **4단계 모달 프로세스**:
  1. 기본 정보 입력 (이름, 연령대, 상태 등)
  2. 경력/전문성 입력 (스킬, 경력 요약)
  3. AI 분석 애니메이션 (모의 로딩 UI를 통해 사용자 경험 극대화)
  4. 결과 생성 (적합도 점수, 추천 직무 등)
- 데이터는 브라우저의 `localStorage` ('kapae_wg')에 저장되어 다른 섹션(Journey 섹션 등)과 연동됩니다.

### 3.3. AI 고용정보 시스템 데모 (`journey_section_v2.js`)
워크넷, 공공기관, 민간(사람인/잡코리아), 기업 직접 채용 등 방대한 소스에서 정보를 수집하여 매칭하는 과정을 시각화한 섹션입니다.
- **동적 인포그래픽**: 숫자가 카운트업 되는 애니메이션 및 실시간 공고 수집 피드(Feed) 시뮬레이션 제공.
- **AI 매칭 데모 UI**: '고용정보 소스 탐색' 탭과 'AI 매칭 결과' 탭을 제공하여, 사용자가 직무 유형별(재취업, 자문, 강의, 프로젝트) 적합도 높은 일자리를 추천받는 프로세스를 체험할 수 있습니다.

### 3.4. 전용 알림마당 커뮤니티 (`notice.html`)
공지사항과 협회 소식을 제공하는 독자적인 페이지입니다.
- **Supabase 연동**: 백엔드 데이터베이스에서 `notice_board` 테이블의 데이터를 읽어와 렌더링. 에러 시 폴백(Fallback) 목업 데이터 제공.
- **경량 관리자 모드(Admin Mode) 내장**:
  - 비밀번호 기반(하드코딩된 인증: `admin.kapae` / `kapae505`)으로 관리자 로그인 성공 시, UI에 관리자 배지와 조작 버튼들이 표시됨.
  - 모달 폼 내부에서 게시물을 **생성, 수정(PATCH), 삭제(DELETE)** 할 수 있는 CUD 기능이 완벽하게 구현됨.
  - WYSWYG 에디터 대신 경량 HTML 주입 툴바 제공.

## 4. 특징 및 개선 제안 (총평)

- **UX/UI 디자인**: 모던한 폰트(Noto Serif KR, Cormorant Garamond 등)와 Glassmorphism, 섬세한 마이크로 애니메이션, 그라디언트를 사용해 공공기관/협회의 신뢰감(Navy)과 프리미엄(Gold/Purple) 분위기를 매우 잘 살렸습니다.
- **구현 방식**: 별도의 번들러 처리 없이 Vanilla JS로 직접 작성되어 있어 직관적이나, 프로젝트 규모가 커질 경우 유지보수(컴포넌트 재사용 등)를 위해 React, Vue.js 혹은 Vite 같은 도구 도입을 고려해 볼 만합니다.
- **보안**: `notice.html`의 관리자 계정/비밀번호(`ADM_ID`, `ADM_PW`)와 Supabase의 Anon Key가 프론트엔드 코드에 평문으로 노출되어 있습니다. 민감한 데이터나 쓰기 권한은 추후 RLS(Row Level Security) 설정이나 백엔드 서버를 통한 안전한 처리가 필요합니다.
